# 🏅 OBI 2026 - Preparação para a Segunda Fase

Este repositório reúne minha resolução das questões da **Olimpíada Brasileira de Informática (OBI) 2026**, utilizadas durante minha preparação para a competição.

O objetivo é documentar meu processo de estudo, armazenar as soluções desenvolvidas em **C++** e disponibilizar os enunciados das questões (quando permitido) organizados junto aos respectivos códigos.

Graças à preparação registrada neste repositório, consegui a classificação para a **segunda fase da OBI 2026**. 🚀

---

# 📂 Estrutura do repositório

Cada pasta corresponde a uma questão, com o código que utilizei para resolve-la e um README.md com um resumo da questão.

Exemplo:

```
📦 OBI-2026
│   ├── 📁 Questão Elevador
│   │   ├── elevador.pdf
│   │   └── elevador.cpp
│   │   └── README.md
|   |
│   ├── 📁 Questão Encontro de Amigas
│   │   ├── encontro.pdf
│   │   └── encontro.cpp
|   |   └── README.md
│   │
│   └── ...
│
└── README.md
```

Cada questão possui:

- 📄 Enunciado em PDF
- 💻 Solução em C++
- 📄 Resumo da questão em md
