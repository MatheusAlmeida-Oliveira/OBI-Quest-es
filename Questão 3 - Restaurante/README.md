# 🍽️ Restaurante

## 📖 Enunciado

O enunciado completo da questão está disponível no arquivo:

📄 **`Restaurante.pdf`**

## 📝 Resumo

Um restaurante possui apenas **mesas com quatro lugares** e recebe reservas de grupos com **1, 2, 3 ou 4 pessoas**.

Pessoas de grupos diferentes podem compartilhar uma mesma mesa, desde que **os integrantes de um mesmo grupo permaneçam juntos**.

A partir da quantidade de grupos de cada tamanho, o objetivo é determinar **o menor número de mesas necessário** para acomodar todos os clientes.

---

## 📥 Entrada

A entrada consiste em uma única linha contendo quatro números inteiros:

* **G₁:** quantidade de grupos com 1 pessoa;
* **G₂:** quantidade de grupos com 2 pessoas;
* **G₃:** quantidade de grupos com 3 pessoas;
* **G₄:** quantidade de grupos com 4 pessoas.

Cada valor representa o número de grupos daquele tamanho que deverão ser acomodados no restaurante.

---

## 📤 Saída

O programa deve imprimir um único número inteiro correspondente à **quantidade mínima de mesas** necessária para acomodar todos os grupos.

## 🎯 Objetivo da questão

Esta questão propõe um problema de organização e alocação de grupos em mesas com capacidade fixa. O desafio consiste em distribuir os grupos de forma que todos sejam acomodados, respeitando as restrições do enunciado e utilizando a menor quantidade possível de mesas.
